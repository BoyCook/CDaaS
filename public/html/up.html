Countdown to events or just time something

/helper

It's easy, navigate to a URL to start your countdown:

  • http://cdaas.co.uk/to/20131225/for/Christmas
  • http://cdaas.co.uk/from/180/for/Egg Boiled
  • http://cdaas.co.uk/up/180/for/Egg Boiled
  • See the full API for all the possibilities

You can also add some parameters to make things interesting:

  • ?msg=Merry Christmas
  • ?warning=30:yellow,10:red
  • See the params section of the docs for all the possibilities
